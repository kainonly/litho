## 安装

```shell
npm install @weplanx/common -S
```

| 扩展库              | 说明         |
| ------------------- | ------------ |
| @weplanx/components | 组件库       |
| @weplanx/settings   | 管理者工具库 |
| @weplanx/resources  | 服务资源库   |
| @weplanx/center     | 个人中心库   |
